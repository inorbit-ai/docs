# InOrbit Connect

The ecosystem of hardware partners that are integrated with InOrbit.

## What is InOrbit Connect?

InOrbit Connect includes:

- **Robot vendors with tight integrations**: Partners like Karcher with deep platform integration
- **Robot OEMs with connectors**: Integrations built for robots like BlueBotics, Gaussium
- **Standards-based robots**: Support for robots using standards like Siemens, MassRobotics AMR, VDA 5050, Open-RMF

## Partner Program

The InOrbit Partner Program includes:

- **Channel partners**: Help with sales process (e.g., Globant, Arrow)
- **Delivery partners**: Provide solutions for end users (e.g., WAKU, Novelte, Robotic Crew)
