description: "RESTful APIs for integrating robot functionality"
---

InOrbit offers REST and streaming APIs that enable programmatic access to InOrbit features. Using these APIs you can fetch data related to your robots, trigger actions, integrate with incident management systems and more.

## Available API Categories

- Robots
- Collections & Tags
- Attributes
- Localization & Maps
- Navigation
- Robot Lock
- Actions
- Incidents
- Mission Tracking
- Push APIs
- Advanced Settings
- Audit Logs

Check the APIs' documentation and specs [here](https://api.inorbit.ai/docs/index.html).
