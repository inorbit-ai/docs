# OpenRobOps

Open source version of fleet management functionality for robot developers who want to build their own on-prem or self-hosted solutions.

## What is OpenRobOps?

OpenRobOps (ORO) is an open source solution that provides most of the functionality in Ground Control. It's designed for:

- **On-prem fleet managers**: Create your own fleet manager to offer customers
- **Self-hosted cloud solutions**: Host your own fleet management platform
- **Fully-branded solutions**: Build a branded fleet manager to complement your robots

## Key Features

- Single-tenant architecture
- Flexible infrastructure choices
- Open source robot-facing code (agent, SDKs, etc.)
- Community-driven development
